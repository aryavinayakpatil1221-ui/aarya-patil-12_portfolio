# Aarya Patil Portfolio

Static portfolio website for Aarya Patil.

## Run locally

1. Open a terminal in the project folder:
   ```powershell
   cd c:\Users\HP\Downloads\final
   ```
2. Start a local server with Python:
   ```powershell
   python -m http.server 8000
   ```
3. Open the site in your browser:
   ```text
   http://localhost:8000
   ```

## Project files

- `index.html` — main portfolio page
- `images/` — profile image asset
- `resume/` — resume PDF asset

## Notes

- The page uses inline CSS and JavaScript.
- The contact form is static and only shows a preview message on submit.
- If port `8000` is already in use, change it to another available port, for example `8001`.
